\textbf{例 75} 将下列直角坐标系下的方程化为极坐标系下的方程.

\begin{enumerate}
    \item \(x^2 + xy + y^2 = 1\);
    \item \(xy = 2\);
    \item \(x^2 + (y - 2)^2 = 4\);
    \item \(x + \sqrt{3}y = 4\);
    \item \(y = e^x\).
    \item $y = e^x$

\end{enumerate}
